环境配置

  • A Package Manager

    Recommendations: WSL for Windows, HomeBrew for Mac or Linux (the one you are most comfortable with)

  • MERN Stack

    （windows 安装在WSL下， Mac使用homebrew安装）

    Can be installed with your package manager

  • NPM packages

    You can stick with npm but yarn is a better option. You may use npm/yarn install these packages globally

    node-modules, .bin

    parcel

  • A Terminal App:

    Recommendations: iTerm2 for mac and Windows Terminal for windows, Cross-platform hyper and alacrity are also worth trying out

  • CLI-Environment (OPTIONAL)

    Zsh with oh-my-zsh/zprezto or fish

  • Browser

    Firefox and Safari or Edge (chromium version) will also do, but Chrome is the defacto standard these days

    • Browser Plugins
  • Text Editor

    Sublime Text and Atom are also good alternatives but VSCode is by the the most widely supported text editor. And you are more than welcomed to use emacs/vim if you are an experienced user.

    • Editor Plugins
